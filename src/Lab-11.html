📊 DoS vs DDoS

  • Denial of Service (DoS): Single attacker overwhelms a target.
  • Distributed Denial of Service (DDoS): Multiple compromised systems (botnet) attack simultaneously.

Attack Categories

Volume-Based Attacks: Saturate bandwidth (UDP flood, ICMP flood, DNS amplification).
Protocol Attacks: Exhaust server resources (SYN flood, Ping of Death, Smurf).
Application-Layer Attacks: Target specific services (HTTP flood, Slowloris, RUDY).

🛡️ DoS/DDoS Mitigation Techniques

📋 Detection Methods

  • Baseline traffic analysis
  • Anomaly detection (spikes in SYN packets, bandwidth)
  • Flow data analysis (NetFlow, sFlow)

🔧 Mitigation Strategies

  • Rate limiting (iptables, Nginx)
  • SYN cookies
  • Anycast network distribution
  • Web Application Firewall (WAF)
  • Cloud-based DDoS protection (Cloudflare, AWS Shield)
