🦠 Types of Malware

Virus: Attaches to clean files; spreads through user action.
Worm: Self-replicating; spreads across networks without user interaction.
Trojan Horse: Disguised as legitimate software; creates backdoors.
Ransomware: Encrypts files and demands payment.
Rootkit: Hides malicious processes and files from detection.
Spyware / Keylogger: Monitors user activity and steals credentials.

📊 Malware Analysis Techniques

  • Static Analysis: Examining code without execution (strings, disassembly, PE headers).
  • Dynamic Analysis: Running malware in a sandbox to observe behavior (file changes, network connections).
  • Behavioral Analysis: Monitoring system calls, registry changes, and process injection.

Common Tools

Tool Purpose
IDA Pro / Ghidra Disassembly / decompilation
Process Monitor Monitor file/registry activity
Wireshark Network traffic analysis
Cuckoo Sandbox Automated malware analysis

⚠️ Malware Indicators of Compromise (IoCs)

  • Unusual outbound network traffic
  • Suspicious registry modifications
  • Disabled security software
  • New startup entries
  • Anomalous process behavior
  • File encryption or renaming
