📜 Essential Terminology

  • Vulnerability: Weakness in a system
  • Threat: Potential danger
  • Risk: Likelihood × Impact
  • Exploit: Code/technique to use vulnerability
  • Payload: Malicious code delivered
  • Zero-day: Unknown vulnerability
  • Penetration Testing: Simulated attack
  • Red Team/Blue Team: Offense/Defense

📊 Interpreting Results

WHOIS Output Analysis

  • Registrar: Company where domain was registered
  • Name Servers: DNS servers authoritative for the domain
  • Creation/Expiration Dates: Useful for social engineering
  • Registrant Contact: Often redacted for privacy (GDPR)

DNS Records (dig)

  • A Record: IPv4 address
  • MX Record: Mail servers
  • TXT Record: Often contains SPF/DMARC policies
  • NS Record: Name servers
