📦 Cloud Service Models

IaaS (Infrastructure as a Service): Virtual machines, storage, networks (e.g., AWS EC2, Azure VMs).
PaaS (Platform as a Service): Development platforms, databases (e.g., Google App Engine, AWS RDS).
SaaS (Software as a Service): Ready-to-use applications (e.g., Office 365, Salesforce).

🌍 Deployment Models

  • Public Cloud: Services over internet (AWS, Azure, GCP).
  • Private Cloud: Dedicated to single organization.
  • Hybrid Cloud: Mix of public and private.
  • Multi-Cloud: Using multiple providers.

⚠️ Common Cloud Threats

  • Misconfigured storage (public S3 buckets, Azure blobs)
  • Insecure APIs and interfaces
  • Credential theft and weak authentication
  • Container escape vulnerabilities
  • Serverless function injection
  • Shadow IT and unauthorized services
