Gain access, escalate privileges, maintain persistence, and cover tracks.

Lab Progress: 0%

📌 System Hacking Phases

  1. Gaining Access: Exploit vulnerabilities to get initial foothold (password cracking, exploits, social engineering).
  2. Escalating Privileges: Elevate from standard user to admin/root.
  3. Maintaining Access: Install backdoors, create new accounts, use rootkits.
  4. Covering Tracks: Clear logs, hide files, disable auditing.

⚡ Common Attack Vectors

  • Password Attacks: Brute-force, dictionary, rainbow tables, pass-the-hash
  • Exploitation: Metasploit, manual exploits for known CVEs
  • Client-side Attacks: Malicious documents, drive-by downloads
  • Post-exploitation: Keylogging, credential dumping (Mimikatz)
