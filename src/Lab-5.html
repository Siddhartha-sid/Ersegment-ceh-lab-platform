🔍 What is Enumeration?

Enumeration is the third phase of ethical hacking. It involves actively connecting to a target system to gather detailed information such as:

  • Valid usernames and group memberships
  • Network shares and resources
  • Running services and their versions
  • Operating system details
  • SNMP community strings
  • DNS zone transfers

Key Difference from Scanning: Enumeration extracts detailed, actionable information by probing services directly.

📡 Common Enumeration Services

NetBIOS (Port 139/445) — Windows shares, users, groups via nbtscan, enum4linux
SMB (Port 445) — File shares, permissions with smbclient, smbmap
SNMP (Port 161/162 UDP) — Network device info using snmpwalk
LDAP (Port 389/636) — Directory services enumeration with ldapsearch
SMTP (Port 25) — User verification via VRFY/EXPN commands
DNS (Port 53) — Zone transfers, subdomain brute-force

⚠️ Important Note

Enumeration often requires valid credentials or misconfigured services. Always obtain written permission before conducting enumeration activities. Unauthorized enumeration may be considered illegal.
