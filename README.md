# Ersegment-ceh-lab-platform
🛡️ CEH v13 Practical Lab Platform – A browser-based cybersecurity training environment for ethical hacking certification. Covers all 20 modules with interactive terminals, simulated tools (Nmap, Burp Suite, Metasploit, etc.), CTF challenges, and quizzes. Built with Flask and vanilla JavaScript. Ideal for instructors, students, and self-learners.
